Contributing and Support
=======================

Community contributions
-----------------------

Contributions to ``geopolrisk-py`` are welcome.

Guidelines for contributing code, reporting bugs, and proposing improvements
are provided in the repository contribution guide:

https://github.com/akoyamp/geopolrisk-py/blob/main/CONTRIBUTING.md

Users are encouraged to follow these guidelines when submitting pull requests
or suggesting changes.

Reporting issues
----------------

Bug reports, feature requests, and reproducible technical issues should be
submitted via the GitHub issue tracker:

https://github.com/akoyamp/geopolrisk-py/issues

Using the issue tracker ensures that problems and discussions are publicly
visible and traceable.
